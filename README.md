# rocm-optimized

Build ROCm that is optimized for your computer

ROCm distributions are built for the general CPU architecture, e.g., AMD64 and AARCH64.
This leaves performance on the table if one has a newer CPU, e.g., Ryzen5 which is in the Strix Halo chipset.

Build custom, optimized ROCm libraries and tools that take full advantage of your PC's power!
